# Phone Number Locator

This a project to search a location of a specified phone number, and locate the map to the phone number location.

![Preview](https://github.com/howardchn/PhoneNumberLocator-JS/raw/master/preview.png)

- Input a cellphone number in the upper left input
- Press `Locate` button
- It will search where location of the phone number and navigate the map to the location automatically.
